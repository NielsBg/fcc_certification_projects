Introduction

What is the console on CS2 and why should you use it?

The console is just that. A console. It's a feature of nearly all of Valve's games that utilize the Source Engine.

  • A way for players to tinker with the game's settings on the fly, without using the User Interface
  • Sometimes getting access to settings that don't even exist in said UI.

    • Even though Counter-Strike 2 uses the newest Source 2 engine, the console feature is still implemented and fully usable. CS2's settings include ALL the customizability you could want, yet some people like myself still prefer to use the console, which can be useful when:

  • Adjusting your own personal settings
  • Setting up a community server for you and your friends to play on.

Let's dive into some of these settings you can learn yourself.

How To Turn The Console On

  • To use the Console, you first have to enable it. In order to do that, navigate to this panel in the Settings and set "Enable Developer Console (~) to 'Yes'

Good job! Now you have access to the console. At any time you can press the "~" button on your keyboard to pull it up.

Media Commands

Let's start with something simple such as the media commands.
Turn on the console.
The first and easiest command we are going to use is "volume". This lets you adjust, you guessed it: the game's volume. A benefit of using this command is that you can tweak the volume more precisely than from the settings. For an example in the settings you can't set the volume to 0.01(0.1 is the minimum value available). volume 1 //Here you can set any value for the volume from 0-1, using '.' i.e '0.05'

Is the enemy team toxic and irritating? Spamming distracting or simply mean messages? Try this command: cl_mute_enemy_team 1 //Set a value from 0(off) to 1(on) - This blocks all communication from the enemy team.
Are your teammates annoying, too loud and don't even speak English? Use this: cl_mute_all_but_friends_and_party 1 //Once again, 0(off), 1(on) - This blocks all communication from everyone that isn't your Steam friend, or from your party.

Crosshair and Viewmodel Commands

Here's a list of some Crosshair commands to actively change your crosshair without opening settings:
cl_crosshairsize [number] //Adjust length of crosshair [-20:20].
cl_crosshairgap [number] //Adjust center gap of crosshair [-10:10].
cl_crosshairthickness [number] //Adjust thickness of crosshair [-2:2].
cl_crosshairstyle [number] //Select different crosshair styles [1:5].
cl_crosshaircolor [number] //Select color of crosshair [0:4].
cl_crosshair_drawoutline [number] //Toggle the outline on or off [0:1].
cl_crosshair_outlinethickness [number] //Adjust thickness of outline [0.1:3].

Viewmodel commands:
viewmodel_fov [number] //Adjust FOV of viewmodel [54:68].
viewmodel_offset_x [number] //Adjust horizontal position of viewmodel [-2.5:2.5].
viewmodel_offset_y [number] //Adjust model distance [-2:2].
viewmodel_offset_z [number] //Adjust vertical position of viewmodel [-2:2].
